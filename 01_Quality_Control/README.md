# Step 1: Quality Control (FastQC)

## Objective
To evaluate the quality of raw paired-end FASTQ files before preprocessing.

## Tool
- FastQC

## Input
- SRR14684620_1.fastq.gz
- SRR14684620_2.fastq.gz

## Command

```bash
fastqc SRR14684620_1.fastq.gz SRR14684620_2.fastq.gz
```

## Output
- FastQC HTML Report
- FastQC ZIP Report
